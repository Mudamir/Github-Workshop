Basic Git Commands

  • git add . Add changed files to the staging area (prepare files for commit).
  • git commit -m "your message" Save a snapshot of your staged changes with a short message.
  • git push Upload your local commits to GitHub (remote repository).
  • git checkout branch-name Switch to another branch to work on a different version.
